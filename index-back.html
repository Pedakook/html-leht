Tere tulemast HTML lehele

Sissejuhatus HTML-i

HTML (Hüperteksti märgistuskeel) on standardne märgistuskeel veebilehtede loomiseks. See määratleb veebilehe struktuuri ja paigutuse, kasutades erinevaid elemente ja märke.

HTML Elemendid

HTML elemendid on HTML lehe ehituskivid. Need määratlevad lehe struktuuri ja sisu. Näiteid HTML elementidest on pealkirjad, lõigud, pildid, lingid ja palju muud.

HTML Atribuudid

HTML atribuudid annavad täiendavat teavet HTML elementide kohta. Neid kasutatakse elemendi käitumise või välimuse muutmiseks. Näiteid HTML atribuutidest on "src" atribuut pildi allika määramiseks ja "href" atribuut lingi sihtkoha määramiseks.

HTML Märksõnad

HTML märksõnad märgistavad sisu HTML dokumendis. Need määratlevad sisu eesmärgi ja tähenduse. Näiteid HTML märksõnadest on "h1" märksõna pealkirjale, "p" märksõna lõigule ja "img" märksõna pildile.
